Two Way Communication for deaf people

Project Description

An innovative assistive framework engineered to remove structural communication barriers between the deaf community and non-sign language users. The application features a dual-module translation core that converts spoken voice into readable text or dynamic sign representations, while translating manual text submissions back into clear synthetic audio outputs.

Technologies Used

PythonOpenCVTensorFlowWebRTCJavaScript API

Real Outcome of the Project

Achieved a high translation accuracy rate for conversational language tokens. The web application establishes a lightweight communication channel operating at minimal latency thresholds, making it highly effective for daily conversational environments.

Usage & Execution Instructions

1. Grant audio recording permissions to authorize language processing streams.
2. Speak directly into the system or input structural text into the interactive dashboard.
3. The communication matrix renders immediate visual components or synthesized audio signals in real time.
