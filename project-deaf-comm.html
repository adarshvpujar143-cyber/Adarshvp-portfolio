Two-Way Communication System for Deaf People

Real-time Sign Language Translation via Computer Vision

🛠️ Technologies Used

Python / JavaScript (core logic) Machine Learning (Gesture recognition models) OpenCV (hand tracking and image processing) TensorFlow / MediaPipe (sign detection and classification) Speech-to-Text API (Google Speech API or similar) Text-to-Speech engine Web or Mobile framework (React / Flask / Android Studio)

⚙️ How It Works

  1. Camera captures hand gestures
  2. ML model classifies sign language gestures into text
  3. System converts text to speech for hearing users
  4. Speech input is converted into text/visual output for deaf users
  5. Optional UI displays real-time conversation panel

🚀 Outcomes

  • Improved accessibility for deaf/mute individuals
  • Real-time bidirectional communication
  • Reduced dependency on interpreters
  • Prototype achieved accurate gesture-to-text conversion in controlled environments

🧠 What I Learned

  • Basics of computer vision and image processing
  • Training and deploying ML models for real-world use
  • Importance of accessibility-focused design
  • Handling real-time data processing and optimization
  • UI/UX design for assistive technology
